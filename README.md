# Octopath Traveler "Strategist" Checker

Back in 2022, I created a quick [node-based gist](https://gist.github.com/iccir/9d7488657512986b61a1eb17adafcad4) to help figure out which monsters I was missing for Octopath Traveler's "Strategist" Achievement.

This repository contains a web-based version to make it more accessible for others.
